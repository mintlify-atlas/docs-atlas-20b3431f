The `tsim.utils` module provides utility functions for circuit visualization, transversal encoding, linear algebra operations, and program text conversion.

## Overview

This module contains:
- **diagram**: SVG diagram rendering for quantum circuits
- **encoder**: Transversal encoder utilities for QEC codes
- **linalg**: Linear algebra utilities for GF(2) operations
- **program_text**: Conversion between tsim shorthand and Stim syntax

## tsim.utils.diagram

SVG diagram rendering for quantum circuits.

### Main Functions

#### render_svg

```python
def render_svg(
    c: stim.Circuit,
    type: str,
    *,
    tick: int | range | None = None,
    filter_coords: Iterable[Iterable[float] | stim.DemTarget] = ((),),
    rows: int | None = None,
    width: float | None = None,
    height: float | None = None,
) -> Diagram
```

Render a Stim circuit timeline/timeslice diagram with custom labels.

Automatically handles custom gate representations:
- `S[T]` → T
- `S_DAG[T]` → T†
- `I[R_Z(theta=0.3*pi)]` → R_Z with annotation
- `I[U3(...)]` → U₃

**Parameters:**
- `c`: Stim circuit to render
- `type`: Diagram type ("timeline-svg" or "timeslice-svg")
- `tick`: Specific tick or range of ticks to display
- `filter_coords`: Coordinates to filter display
- `rows`: Number of rows in timeslice diagram
- `width`: Explicit width for container
- `height`: Desired height

**Returns:** `Diagram` object with SVG markup

**Example:**
```python
import stim
from tsim.utils.diagram import render_svg

circuit = stim.Circuit("""
    H 0
    S[T] 0
    CNOT 0 1
    I[R_Z(theta=0.3*pi)] 1
    M 0 1
""")

diagram = render_svg(circuit, "timeline-svg", height=200)
# Display in Jupyter notebook automatically
display(diagram)
```

## tsim.utils.linalg

Linear algebra utilities for GF(2) operations.

### find_basis

```python
def find_basis(vectors: np.ndarray) -> tuple[np.ndarray, np.ndarray]
```

Decompose binary vectors into a basis subset and transformation matrix over GF(2).

Given vectors V, finds a maximal linearly independent subset B (the basis) and computes transformation matrix T such that:

```
V = T @ B (mod 2)
```

**Algorithm:** Gaussian elimination in GF(2)
1. For each vector v:
   - Try to reduce v using existing basis vectors
   - If v reduces to 0, it's dependent (record linear combination)
   - If v reduces to non-zero, it's independent (add to basis)
2. Build transformation matrix from linear combinations

**Parameters:**
- `vectors`: Input binary vectors of shape `(N, D)`, elements 0 or 1

**Returns:** Tuple `(basis, transform)`
- `basis`: Subset of independent vectors, shape `(K, D)` where K is rank
- `transform`: Transformation matrix, shape `(N, K)`

**Example:**
```python
import numpy as np
from tsim.utils.linalg import find_basis

# Three vectors, third is XOR of first two
vectors = np.array([
    [1, 0, 1, 0],
    [0, 1, 1, 0],
    [1, 1, 0, 0],  # XOR of first two
], dtype=np.uint8)

basis, transform = find_basis(vectors)
# basis.shape = (2, 4) - only two independent vectors
# transform.shape = (3, 2)
# transform[2] = [1, 1] - third vector is sum of first two

# Verify reconstruction
reconstructed = (transform @ basis) % 2
assert np.array_equal(reconstructed, vectors)
```

## tsim.utils.program_text

Conversion utilities between tsim shorthand and Stim program text.

### shorthand_to_stim

```python
def shorthand_to_stim(text: str) -> str
```

Convert tsim shorthand syntax to valid Stim instructions.

**Conversions:**
- `T 0 1` → `S[T] 0 1`
- `T_DAG 0 1` → `S_DAG[T] 0 1`
- `R_Z(0.3) 0` → `I[R_Z(theta=0.3*pi)] 0`
- `R_X(0.25) 0` → `I[R_X(theta=0.25*pi)] 0`
- `R_Y(-0.5) 0` → `I[R_Y(theta=-0.5*pi)] 0`
- `U3(0.3, 0.24, 0.49) 0` → `I[U3(theta=0.3*pi, phi=0.24*pi, lambda=0.49*pi)] 0`

**Example:**
```python
from tsim.utils.program_text import shorthand_to_stim

shorthand = """
H 0
T 0
R_Z(0.25) 1
CNOT 0 1
"""

stim_text = shorthand_to_stim(shorthand)
print(stim_text)
# Output:
# H 0
# S[T] 0
# I[R_Z(theta=0.25*pi)] 1
# CNOT 0 1
```

### stim_to_shorthand

```python
def stim_to_shorthand(text: str) -> str
```

Convert expanded Stim annotations back to tsim shorthand.

**Conversions:**
- `I[U3(theta=θ*pi, phi=φ*pi, lambda=λ*pi)]` → `U3(θ, φ, λ)`
- `I[R_X(theta=α*pi)]` → `R_X(α)`
- `I[R_Y(theta=α*pi)]` → `R_Y(α)`
- `I[R_Z(theta=α*pi)]` → `R_Z(α)`
- `S[T]` → `T`
- `S_DAG[T]` → `T_DAG`

**Example:**
```python
from tsim.utils.program_text import stim_to_shorthand

stim_text = """
H 0
S[T] 0
I[R_Z(theta=0.25*pi)] 1
CNOT 0 1
"""

shorthand = stim_to_shorthand(stim_text)

# H 0
# T 0
# R_Z(0.25) 1
```

### Example 3: Linear Algebra for Error Correction

```python
import numpy as np
from tsim.utils.linalg import find_basis

# Stabilizer generators for a simple code
stabilizers = np.array([
    [1, 1, 0, 0],  # X1 X2
    [0, 0, 1, 1],  # Z3 Z4
    [1, 1, 1, 1],  # X1 X2 Z3 Z4 (product of first two)
], dtype=np.uint8)

basis, transform = find_basis(stabilizers)
print(f"Rank: {len(basis)}")
print(f"Independent generators:\n{basis}")
print(f"Transformation matrix:\n{transform}")
```

### Example 4: Program Text Conversion

```python
from tsim.utils.program_text import shorthand_to_stim, stim_to_shorthand
import tsim

# Write circuit in shorthand

U3(0.5, 0.25, 0.75) 0
```

### Example 5: Custom Transversal Encoder

```python
from tsim.utils.encoder import TransversalEncoder

class CustomEncoder(TransversalEncoder):
    def __init__(self):
        # Define custom [[n, k, d]] code
        encoding_program = """
        # Custom encoding circuit
        R 0 1 2
        H 0
        CNOT 0 1
        CNOT 0 2
        """
        
        super().__init__(
            n=3,  # 3 physical qubits
            encoding_qubit=0,
            encoding_program_text=encoding_program,
            logical_gate_expansions={
                # Define how logical gates map to physical gates
                "H": ["H"],
            },
            stabilizer_generators=[[0, 1], [1, 2]],
            observables=[[0]],
        )

encoder = CustomEncoder()
encoder.initialize("R 0")
encoder.encode_transversally("H 0\nM 0")
```

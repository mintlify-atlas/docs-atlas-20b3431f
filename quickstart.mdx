tsim is a GPU-accelerated quantum circuit sampler that feels just like [Stim](https://github.com/quantumlib/Stim), but supports non-Clifford gates through ZX-calculus stabilizer rank decomposition.

## Your First Circuit

Let's create a simple quantum circuit with Clifford gates, non-Clifford gates, and noise:

<Steps>
  <Step title="Import tsim">
    ```python
    import tsim
    ```
  </Step>
  
  <Step title="Create a circuit">
    Create a circuit using Stim's program text format:
    
    ```python
    circuit = tsim.Circuit("""
        H 0          # Hadamard on qubit 0
        T 0          # T gate (non-Clifford!)
        CNOT 0 1     # Controlled-NOT
        M 0 1        # Measure both qubits
    """)
    ```
    
    <Info>
    tsim supports all [Stim gates](https://github.com/quantumlib/Stim/wiki/Stim-v1.9-Gate-Reference) plus non-Clifford gates like `T`, `T_DAG`, `R_X`, `R_Y`, `R_Z`, and `U3`.
    </Info>
  </Step>
  
  <Step title="Compile the circuit">
    Compile the circuit into a sampler:
    
    ```python
    sampler = circuit.compile_sampler()
    ```
    
    <Note>
    Compilation may take a moment as tsim builds the ZX-calculus representation and performs stabilizer rank decomposition.
    </Note>
  </Step>
  
  <Step title="Sample measurements">
    Generate samples from the circuit:
    
    ```python
    samples = sampler.sample(shots=100)
    print(samples)
    # Output: 2D array of shape (100, 2) with measurement outcomes
    ```
  </Step>
</Steps>

## Complete Example

Here's a complete example showing tsim's key features:

```python
import tsim

# Create a circuit with Clifford gates, non-Clifford gates, and noise
c = tsim.Circuit("""
    H 0
    T 0                      # Non-Clifford T gate
    CNOT 0 1
    PAULI_CHANNEL_1(0.1, 0.1, 0.2) 0 1  # Pauli noise
    DEPOLARIZE2(0.01) 0 1    # Two-qubit depolarizing noise
    M 0 1                    # Measure qubits
    DETECTOR rec[-1] rec[-2] # Define a detector
""")

# Compile and sample measurements
measurement_sampler = c.compile_sampler()
measurements = measurement_sampler.sample(shots=100)

print(f"Measurement samples shape: {measurements.shape}")
print(f"First 5 samples:\n{measurements[:5]}")
```

## Sampling Detectors

For quantum error correction, you'll often want to sample detectors instead of raw measurements:

```python
import tsim

# Circuit with detectors
c = tsim.Circuit("""
    X 0
    M 0
    DETECTOR rec[-1]
""")

# Compile detector sampler
detector_sampler = c.compile_detector_sampler()
detector_samples = detector_sampler.sample(shots=100)

print(detector_samples)
```

<Warning>
tsim reports actual detector samples, not detection flips! In the example above, tsim returns `True` values because the X gate flips the qubit before measurement. Stim would return `False` (no flip from the noiseless reference).
</Warning>

## Using Non-Clifford Gates

tsim's superpower is supporting non-Clifford gates efficiently:

<CodeGroup>
```python T and T_DAG gates
import tsim

c = tsim.Circuit("""
    H 0
    T 0       # T gate: π/4 phase rotation
    T_DAG 1   # T† gate: -π/4 phase rotation
    M 0 1
""")

sampler = c.compile_sampler()
samples = sampler.sample(shots=100)
```

```python Rotation gates
import tsim

c = tsim.Circuit("""
    R_X(0.5) 0   # Rotate around X by π/2
    R_Y(0.25) 1  # Rotate around Y by π/4
    R_Z(1.0) 2   # Rotate around Z by π
    M 0 1 2
""")

sampler = c.compile_sampler()
samples = sampler.sample(shots=100)
```

```python U3 gate
import tsim

# General single-qubit unitary
c = tsim.Circuit("""
    U3(0.5, 0.25, 0.125) 0
    M 0
""")

sampler = c.compile_sampler()
samples = sampler.sample(shots=100)
```

```python Programmatic construction
import tsim

# Build circuits programmatically
c = tsim.Circuit()
c.append("H", 0)
c.append("T", 0)
c.append("CNOT", [0, 1])
c.append("M", [0, 1])

sampler = c.compile_sampler()
samples = sampler.sample(shots=100)
```
</CodeGroup>

<Tip>
Rotation gate parameters are specified as multiples of π. For example, `R_X(0.5)` rotates by 0.5π = π/2.
</Tip>

## Migrating from Stim

If you have existing Stim code, tsim is designed to be a drop-in replacement:

```python
# Before (Stim)
import stim
c = stim.Circuit("...")
sampler = c.compile_sampler()
samples = sampler.sample(100)

# After (tsim)
import tsim
c = tsim.Circuit("...")  # Same syntax!
sampler = c.compile_sampler()
samples = sampler.sample(100)
```

The main difference: tsim supports non-Clifford gates and returns actual detector samples (not flips).

## Circuit Properties

Query useful information about your circuit:

```python
import tsim

c = tsim.Circuit("""
    H 0
    T 0
    CNOT 0 1
    M 0 1
    DETECTOR rec[-1] rec[-2]
""")

print(f"Number of qubits: {c.num_qubits}")
print(f"Number of measurements: {c.num_measurements}")
print(f"Number of detectors: {c.num_detectors}")
print(f"T-count: {c.tcount()}")
```

## Random Seeds

For reproducible sampling, specify a random seed:

```python
import tsim

c = tsim.Circuit("""
    H 0
    CNOT 0 1
    M 0 1
""")

# Reproducible sampling
sampler = c.compile_sampler(seed=42)
samples1 = sampler.sample(shots=100)
samples2 = sampler.sample(shots=100)

# samples1 and samples2 will be different, but
# re-creating the sampler with seed=42 gives the same sequence
sampler2 = c.compile_sampler(seed=42)
samples3 = sampler2.sample(shots=100)
# samples3 == samples1 ✓
```
